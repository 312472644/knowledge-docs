##### 1、使其a == 1 && a == 2 && a == 3 成立。

```javascript
主要考虑的是隐式转化。 
const obj = {
      value: 1,
      valueOf() {
        return this.value++;
      },
  };
  console.log(obj == 1 && obj == 2 && obj == 3);
```

##### 3、不用循环和数组内置方法，怎么给数组求和？

```javascript
function sum(array) {
      function f(i) {
        return i >= array.length ? 0 : array[i] + f(i + 1);
      }
      return f(0);
}
```
